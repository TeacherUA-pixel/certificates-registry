# Реєстр сертифікатів підвищення кваліфікації (Firebase Firestore + Google Sites)

Даний проект містить повну систему обробки розрізнених Google Таблиць за різні роки, їх автоматичної консолідації у єдиний формат та завантаження в безкоштовну базу даних **Firebase Firestore (Spark Plan)** з інтеграцією в **Google Sites**.

---

## 📁 Структура проекту

```
google-certificates-page/
├── index.html                    # 3D Scroll Landing & Реєстр з підключенням до Firebase
├── firestore.rules               # Правила безпеки для Firebase Firestore
├── README.md                     # Ця інструкція
├── data/                         # Сюди кладуться вивантажені файли Excel / CSV
├── output/                       # Сюди зберігаються об'єднані JSON файли
└── scripts/
    ├── consolidate_sheets.py     # Скрипт очищення, нормалізації та об'єднання таблиць
    └── upload_to_firestore.py    # Скрипт завантаження даних у Firebase Firestore
```

---

## 🛠 Крок 1: Консолідація та обробка Google Таблиць

1. Експортуйте ваші Google Таблиці за різні роки у формат **CSV** або **Excel (`.xlsx`)**.
2. Скопіюйте їх у папку `data/`.
3. Запустіть скрипт консолідації через термінал:

```bash
python scripts/consolidate_sheets.py
```

### ⚡ Що зробить скрипт автоматично:
* **Авто-визначення колонок**: розпізнає будь-які назви стовпчиків (*ПІБ, П.І.Б., Учасник*, *Сертифікат №, Код, Номер*, *Дата видачі* тощо).
* **Очищення та стандартизація ПІБ**: видаляє зайві пробіли, форматує регістр букв.
* **Форматування дат**: зводить всі дати до стандарту `ДД.ММ.РРРР`.
* **Збереження**: створює готові консолідовані файли `courses.json` та `certificates.json` у папці `output/`.

---

## 🔥 Крок 2: Налаштування та завантаження в Firebase (Безкоштовний тариф)

1. Перейдіть на [Firebase Console](https://console.firebase.google.com/) та створіть новий проект (безкоштовний Spark Plan).
2. Створіть **Cloud Firestore Database** в тестовому режимі.
3. У **Project Settings** -> **Service Accounts** натисніть **Generate new private key** та збережіть завантажений файл під назвою `serviceAccountKey.json` у папку `scripts/`.
4. Запустіть завантаження даних у Firebase:

```bash
python scripts/upload_to_firestore.py
```

---

## 🌐 Крок 3: Підключення веб-сторінки до Firebase & Google Sites

1. У Firebase Console перейдіть до **Project Settings** -> **General** -> **Your apps** -> створіть Web App (`</>`).
2. Скопіюйте ваш `firebaseConfig` та вставте його у файл **`index.html`** (на початку тегу `<script type="module">`).
3. Скопіюйте весь код файлу **`index.html`** та вставте його у ваш **Google Sites** через кнопку **Вставити (Embed)** -> **Вставити код**.

## ✨ Переваги цього рішення:
- **0 грн витрат**: працює 100% у межах безкоштовного тарифу Firebase Spark.
- **Швидкість**: миттєва перевірка справжності за 1 точковий запит до бази.
- **Автоматизація**: єдиний скрипт зводить десятки таблиць будь-яких років.
- **Преміальний дизайн**: 3D-скрол лендінг у стилі брендбуку Google.
